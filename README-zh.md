# react-components

[示例](https://fxss5201.github.io/react-components/)

`react ^19.2.0` + `react-router ^7.9.6` + `react-redux ^9.2.0` + `antd ^6.1.0` + `typescript ~5.8.3`

通过配置既可作为展示网站、又可作为后台管理系统。

## 核心功能

1. 主题切换
2. 多语言
   1. 支持通过命令行导出（excel/json/md）、导入(excel/json/md)多语言文件
3. 路由中间件
4. 路由对象可配置 是否缓存 、权限、水印及以下布局调整：
   1. 在菜单中是否隐藏该路由
   2. 在标签页中是否隐藏该路由
   3. 进入该路由时是否隐藏头部
   4. 进入该路由时是否隐藏菜单
   5. 进入该路由时是否收起菜单
   6. 进入该路由时是否隐藏底部
   7. 进入该路由时是否隐藏面包屑导航
   8. 进入该路由时是否隐藏标签页
5. MOCK 接口
6. 封装多个组件：
   1. 复制到剪贴板
   2. 文件相关：
      1. 文件选择（支持选择文件、选择文件夹（含根目录）、选择文件夹（不含根目录））
      2. 文件拖拽选择
      3. 文件抽屉（支持上传文件/文件夹、下载文件/文件夹）
      4. 文件哈希
   3. Markdown 渲染/编辑
   4. JSON 渲染/编辑
   5. 代码编辑器
   6. MDX 渲染
   7. Echarts 封装
   8. 表单块，支持通过配置生成表单。
   9. 表格页，支持通过配置生成表格、表格搜索、表格分页、表格操作弹窗。
   10. 表格页再封装，将通用操作（查询、添加、删除、修改、分页逻辑）封装在一起。
   11. 登录页
   12. 重置密码页
