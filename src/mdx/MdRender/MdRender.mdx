# MdRender

MdRender 组件用于展示 Markdown 内容，内置代码高亮、复制代码功能。

> **注意：**
> - 要在组件内使用目录功能，需要在 Markdown 内容中添加 `## 目录` 或者 `## TOC`，组件会自动解析目录，**每个页面保证只有一个目录**。

## API

### 参数 Props

``` ts
export type MdRenderProps = {
  children: string
  className?: string
  enableCopy?: boolean
}
```

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `children` | Markdown 内容 | `string` | - |
| `className` | 自定义类名 | `string` | - |
| `enableCopy` | 是否开启复制代码功能 | `boolean` | `true` |
